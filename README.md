## Overview

### The challenge

ユーザーができること:

- デバイスの画面サイズに応じて、サイトに最適なレイアウトを表示します
- ページ上のすべてのインタラクティブ要素のホバー状態を確認できる
- コンタクトフォームが送信された時、以下のエラーがある場合エラーメッセージを受け取る:
  - 名前、メールアドレス、メッセージ内容が空白の時
  - メールアドレスのフォーマットが正しくない時
